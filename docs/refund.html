退款政策

最后更新：2026 年 7 月 13 日

一句话总结

PiPanel 的所有付费功能都提供完整的 7 天免费试用，可以在付费前充分确认软件是否适合你的设备和工作流。因此，购买后我们不提供退款。

为什么这样设计

PiPanel 目前只支持 Apple Silicon（M 系列芯片）的 Mac，不同设备、不同工作流下的使用体验也会有差异。与其在购买后处理退款，我们选择把这段确认时间放在购买之前：7 天试用期内可以完整体验所有进阶设置，确认权限、兼容性和实际效果都符合预期后再付费。

购买前的建议

  • 下载 App 并完整试用 7 天，包括你实际会用到的自定义设置。
  • 确认「屏幕录制」「辅助功能」两项权限能正常授权，且不会被你所在环境的管理策略限制。
  • 确认你的 Mac 是 Apple Silicon 机型（Intel Mac 暂不支持）。
  • 如果有任何疑问，欢迎在购买前通过邮件或 GitHub Issue 咨询。

例外情况

如果你遇到重复扣款、支付渠道的技术性错误，或授权码始终无法激活等非试用期能规避的问题，请发邮件到 support@pipanel.app 说明情况，我们会酌情处理。

支付与授权处理方

购买交易通过第三方支付与授权处理商 Creem 完成，具体扣款渠道、账单方式以其结算页面为准。

联系方式

如果你对本退款政策有任何疑问，欢迎邮件联系 support@pipanel.app，或在 GitHub Issues 中提出。
