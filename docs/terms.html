软件说明

PiPanel 是一款免费的 macOS 菜单栏应用，可以把任意窗口变成可移动、缩放、交互的画中画悬浮窗。本软件不隶属于 Apple Inc.，也未获得其官方认可或赞助。

系统权限与使用风险

本软件需要「屏幕录制」和「辅助功能」两项系统权限才能正常工作，这意味着它有能力读取其他应用窗口的画面、移动/缩放其他应用的窗口，并代替你向其他应用发送点击、滚动、按键等操作。这是实现画中画交互功能所必需的，但也意味着你需要对授予这些权限保持基本的知情和信任。请只从官方 GitHub Releases 下载本软件，不要使用来路不明的第三方分发版本。
