Features

不是截图，也不只是镜像

PiPanel 移动并持续运行真实窗口，再把它变成一个真正能操作的悬浮画中画。

真实窗口持续运行

进入画中画的不是一张截图。原软件仍在正常运行、持续刷新，视频、终端、聊天和任务进度都能实时显示。

直接交互，不只是观看

鼠标悬停即可接管操控，点击、滚动、快捷键和键盘输入直接作用于原窗口，像使用普通窗口一样自然。

一甩即成画中画

把窗口甩向屏幕边缘即可自动进入画中画，也可以从菜单栏精确选择，全程不需要整理桌面或反复切换应用。

多窗口与智能堆叠

专业版可同时运行多个画中画，一键堆叠、摊开或全部关闭；空闲时还能自动贴边收起，需要时随手唤回。

工作流不中断

操作画中画后自动归还键盘焦点；切回源软件时自动隐藏画中画，让临时查看和操作不打断当前工作。

画质、性能与外观可调

自由调整帧率、清晰度、分辨率、默认尺寸、圆角、阴影、边框和堆叠方式，在流畅度、功耗与观感之间自行取舍。

How it works

三步，继续使用任何窗口

1

选中窗口

从菜单栏点选，或直接把窗口拖向屏幕边缘触发甩动手势。

2

自动悬浮

真实窗口持续运行，并以画中画形式悬浮显示，可跨 Space、覆盖全屏应用之上。

3

随手操作

鼠标悬停即可交互，拖拽移动、边缘缩放，用完随手甩开即可关闭。

FAQ

常见问题

PiPanel 和普通画中画或窗口镜像有什么不同？

普通画中画通常只能播放视频，窗口镜像通常只能看、不能操作。PiPanel 处理的是正在运行的真实窗口：原软件会持续刷新，你也可以直接在画中画里点击、滚动和输入，操作会实时作用于原窗口。

PiPanel 完全免费吗？

可以永久免费使用一个真实窗口的可交互画中画和开机启动。首次安装还可完整试用专业版 7 天，包括无限画中画、智能堆叠、全局快捷键、画质控制、自动化和完整外观设置；试用结束后，免费核心功能仍然可以继续使用。

7 天专业版试用如何计算？

首次安装并启动 PiPanel 后自动开始计算，7 天内无需付款或绑定支付方式即可使用全部专业版功能。试用到期不会自动扣费，只会恢复为永久免费版；购买后再永久解锁专业版。

授权码怎么获取，在哪里激活？

购买成功后，授权码会发送到你的付款邮箱。打开 App 的「设置 → 会员」，粘贴授权码即可激活对应数量的设备。

一个授权码能在几台设备上使用？

购买时按设备数量选择套餐：单设备（1 台）、双设备（2 台）或多设备（3 台及以上）。同一授权码可以在对应数量的 Mac 上同时激活。

换了新 Mac 怎么办？

在旧设备的「设置 → 会员 → 设备管理」里移除该设备，就能腾出一个激活名额给新 Mac，全程在 App 内自助完成，不需要联系开发者。

买断之后还能获得后续更新吗？

可以。一次性购买包含未来所有版本的更新，不需要额外付费，也没有单独的"升级费"。

需要开启哪些系统权限？

需要「屏幕录制」和「辅助功能」两项权限——前者用于捕获窗口画面，后者用于把点击、滚动、打字等操作转发到源窗口。首次使用时 App 会引导你逐步开启。

支持 Intel Mac 吗？

目前仅支持搭载 Apple Silicon（M 系列芯片）的 Mac，暂不支持 Intel 机型。

最低系统要求是什么？

macOS 14.0（Sonoma）及以上版本，搭配 Apple Silicon 芯片。
