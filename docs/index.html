Features

该有的都有，不该有的一个没有

专注做好一件事：让任意窗口变成一个真正可用、可交互的悬浮小窗。

一甩即成画中画

把窗口拖到屏幕边缘松手，识别甩动手势，自动变成一个悬浮的小窗口，全程无需菜单操作。

直接交互，不只是镜像

鼠标悬停即可接管操控，点击、滚动、打字直接作用在源窗口上，感觉就像在用一个真实的小窗口。

尺寸自适应

拖拽画中画边缘调整大小时，内部软件会尽量跟着一起变化，而不是简单粗暴地裁切或拉伸画面。

自动归还焦点

操作完成后可以自动把键盘焦点还给你之前在用的应用，不打断正在进行的工作流。

可自定义外观

圆角大小、阴影开关、默认尺寸、堆叠角落——独立设置窗口里都能调，不是一套写死的样式。

完全开源

核心功能代码完全公开在 GitHub 上，欢迎阅读、提 issue、发 PR，一起把它做得更好用。

How it works

三步，把任何窗口变成画中画

1

选中窗口

从菜单栏点选，或直接把窗口拖向屏幕边缘触发甩动手势。

2

自动悬浮

窗口以画中画形式悬浮显示，可跨 Space、覆盖全屏应用之上。

3

随手操作

鼠标悬停即可交互，拖拽移动、边缘缩放，用完随手甩开即可关闭。

FAQ

常见问题

PiPanel 完全免费吗？

是的，PiPanel 完全免费、开源，没有隐藏付费项或试用期限制。

需要开启哪些系统权限？

需要「屏幕录制」和「辅助功能」两项权限——前者用于捕获窗口画面，后者用于把点击、滚动、打字等操作转发到源窗口。首次使用时 App 会引导你逐步开启。

支持 Apple Silicon 和 Intel 芯片吗？

支持，构建产物是通用二进制（arm64 + x86_64），两种芯片的 Mac 都可以直接运行。

最低系统要求是什么？

macOS 14.0（Sonoma）及以上版本。
