隐私政策

最后更新：2026 年 7 月 13 日

一句话总结

AnyPiP 不收集、不上传、不与任何人分享你的任何数据。它是一个完全本地运行的 macOS 应用，没有账号系统、没有统计分析、没有崩溃上报、不联网。你可以在 GitHub 上查看完整源代码来验证这一点。

App 需要哪些系统权限，用来做什么

AnyPiP 在首次使用时会请求以下两项 macOS 系统权限，两者都只在你的 Mac 本地生效，不会把任何内容发送到外部服务器：

  • 屏幕录制（Screen Recording）：用于通过 ScreenCaptureKit 捕获你选中窗口的画面，并在画中画悬浮窗里实时显示。捕获到的画面只在本机内存中处理和渲染，不会被保存成文件，也不会被上传。
  • 辅助功能（Accessibility）：用于读取/调整源窗口的位置和尺寸，并把你在画中画悬浮窗里的点击、滚动、按键操作转发到源窗口上。

本地存储的设置

AnyPiP 会把你的偏好设置（例如画面帧率、默认尺寸、外观选项等）保存在 macOS 标准的本地偏好设置（UserDefaults）中，仅存在于你自己的 Mac 上，不会同步到任何账号或云端。

网络请求

AnyPiP 的开源版本不发起任何网络请求，没有统计分析（analytics）、没有崩溃报告工具、没有广告 SDK，也没有任何形式的遥测（telemetry）。

关于这个网站

本网站（lyle-xub.github.io/AnyPiP）托管在 GitHub Pages 上，本身不包含任何统计分析脚本。GitHub 作为托管方可能会按照其自身的隐私政策记录基础的访问日志（例如 IP 地址），这部分由 GitHub 官方隐私声明约束，与开发者无关。

联系方式

如果你对本隐私政策有任何疑问，欢迎在 GitHub Issues 中提出。

AnyPiP 是一个个人开源项目，与 Apple 官方无关。
